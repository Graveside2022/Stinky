ATAK Setup Instructions

Instructions

ATAK Setup

  1. Install ATAK from the official website or app store
  2. Open ATAK and go to Settings > Network > TAK Server
  3. Add a new server with the following details:
    • Server Address: 10.42.0.1
    • Port: 8089
    • Username: admin
    • Password: admin

Configuration

Connection Settings

  1. Enable "Auto Connect" in the server settings
  2. Set "Connection Type" to "WiFi"
  3. Enable "SSL/TLS" for secure connection
  4. Set "Keep Alive" to 30 seconds
  5. Enable "Location Sharing" in ATAK settings

Additional Info

Important Notes

  1. Make sure your device is connected to the same network as the server
  2. GPS must be enabled on your device for location sharing
  3. ATAK will automatically reconnect if the connection is lost
  4. Use the "Disconnect" button in ATAK to manually disconnect from the server
